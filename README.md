# Hirt & Carter — `.github` Repository

This is the default community health and organization profile repository for the [Hirt-And-Carter](https://github.com/Hirt-And-Carter) GitHub organization.

## Contents

| Path | Purpose |
|---|---|
| `profile/README.md` | Public organization profile — displayed on the [Hirt-And-Carter GitHub page](https://github.com/Hirt-And-Carter) |

## About This Repository

The `.github` repository is a special repository recognized by GitHub. Files placed here apply organization-wide defaults and configure the public-facing profile of the organization.

### `profile/README.md`

The file at `profile/README.md` is the **public organization profile README**. It is displayed prominently on the organization's GitHub profile page and is visible to anyone who visits [github.com/Hirt-And-Carter](https://github.com/Hirt-And-Carter).

Content is sourced from the official [Hirt & Carter website](https://hirtandcarter.co.za/) and covers:
- Company overview and vision
- Local and Pan-African footprint stats
- Service offerings
- The RMS+ technology platform

## Contributing

To update the organization profile, edit `profile/README.md` and open a pull request.
